# Shopping Cart - Testtask

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm/yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/alexk176/testtask.git
cd shopping-cart
```

2. Install dependencies:
```basha
pnpm install
```

3. Start the development server:
```bash
pnpm dev
```

4. Open your browser at `http://localhost:5173`
